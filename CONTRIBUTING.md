# Contributing to Legion K Player

We welcome contributions from the community.

## Development Guidelines
- Ensure all code is modular and placed in the appropriate directory (`core/`, `ui/`, `bridge/`, `network/`).
- **Strictly No Comments**: We follow a pure code philosophy. Your code should be self-documenting through clear naming and structure.
- Maintain the 'Blue' aesthetic in all UI modifications.

## Process
1. Fork the repo.
2. Create a feature branch.
3. Commit changes (no comments).
4. Push and open a PR.
