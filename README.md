## What It Is

![Code Matrix](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNXpzN3l4c2N3aXUxZm9uNjFxcGZpdjE1NXo3NzJqdzZqcWFhZ3lsZiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/ule4vhcY1xEKQ/giphy.gif)

Legion K Player is an Android emulator frontend and VM core. It's not trying to be beginner-friendly. It's trying to be fast, deep, and configurable past the point where most emulators tap out.

![System Boot](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExZzFsZTZlMm1rMHBpZjY4aGFuc2cwY21mNGV5bHd1dnE0cWo1ZW11YyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/xT9IgG50Lg7rusyiqU/giphy.gif)

It ships with over 100 modules. That number isn't marketing — each module is a separate implementation covering a specific domain. Gaming, system-level control, network routing, media capture, VM internals. All of it in one build.

![Loading](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNnUwcTBsOHY5MXZneDhzMjFsMW1rdXEwaWNpaW80MHkzZ3NxYm81NiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/du3J3cXyzhj75IOgvA/giphy.gif)

### <img src="https://img.icons8.com/fluency/24/controller.png" width="18"/> Gaming & Performance

![Gaming Reaction](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExczN0a3FxMXZzMGc5Z2I3Mm54NjFvbzlzOGJsb29zaHNxbDBrYzl2YiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/KAq5w47R9rmTuxXUOs/giphy.gif)

**Macro Recorder & Editor**

Records input sequences and replays them on demand. You can build layered scripts — not just button presses, but timed sequences with delays, conditionals, and loops. Useful for farming runs, login automation, anything you'd otherwise do manually a hundred times.

![Macro Demo](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNXpzN3l4c2N3aXUxZm9uNjFxcGZpdjE1NXo3NzJqdzZqcWFhZ3lsZiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/ule4vhcY1xEKQ/giphy.gif)

---

**Multi-Instance Sync**

Runs multiple independent instances in parallel and syncs them. Each one isolated at the VM level — different accounts, different sessions, no bleed. Scale up to dozens without them interfering with each other.

![Multi Instance](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExZjFkbWd0eHh2MHN5cmIyb245aHdxdGl6cGVld3c0MXB2bzRlemJzMCZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/26tn33aiTi1jkl6H6/giphy.gif)

---

**Eco Mode & FPS Booster**

Two modes that work together. Eco cuts background resource drain while FPS Booster pushes rendering priority to the foreground. Net result: stable frames without your CPU running at 100% on a game that shouldn't need it.

![FPS Counter](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExaXkzZDJidDZya3Y5N2k4eGh0NDVlN3Z3YzlsdDRleTcyZG9oMGdzYyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/xT9IgG50Lg7rusyiqU/giphy.gif)

---

**Smart Cast & Skill Combos**

Pre-programs skill chains and fires them on a single trigger. Built for MOBA and action titles where combo timing is the difference between winning and watching the respawn screen. You set the sequence once, it executes it exactly.

![Skill Combo](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNXl4MW54aXZhZmphNHQzMndqaGEycXE5cXRhcHhzdm1sZm1ieGV2eSZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/QvMlVkJ3XSSj9cOxDM/giphy.gif)

---

**XPack & Graphics Overdrive**

Pushes rendering output past the defaults the app expects. More texture detail, better frame compositing, no API cap respected. If the game has headroom, Overdrive finds it.

![Graphics](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExZzFsZTZlMm1rMHBpZjY4aGFuc2cwY21mNGV5bHd1dnE0cWo1ZW11YyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/3oEjHV0z8S7WM4MwnK/giphy.gif)

### <img src="https://img.icons8.com/fluency/24/settings.png" width="18"/> System & Core

![System Deep](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExcDJ6dWx6M3ZtNXUyNTZ6d2J4b3VwYnF5d3ZnNXZ4NHBtemhwdG9zeSZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/077i6AULCXc0FKTj9s/giphy.gif)

The core modules are where Legion K differs from surface-level emulators. This isn't a wrapper with a nice UI. It reaches into hardware.

![Deep Dive](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNnUwcTBsOHY5MXZneDhzMjFsMW1rdXEwaWNpaW80MHkzZ3NxYm81NiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/du3J3cXyzhj75IOgvA/giphy.gif)

**Memory Optimizer**

Watches RAM allocation in real time and reclaims it when processes hold onto memory they're not using. No scheduled cleanup, no manual flush — it runs continuously and keeps the footprint tight.

![RAM](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNXpzN3l4c2N3aXUxZm9uNjFxcGZpdjE1NXo3NzJqdzZqcWFhZ3lsZiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/ule4vhcY1xEKQ/giphy.gif)

---

**Kernel Bridge**

Bypasses the abstraction layer most emulators sit behind. Talks directly to hardware for acceleration. This is why Legion K runs faster on the same machine than most alternatives — it's not doing extra translation work.

![Kernel](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExaTZyMGo5cWZhZnB3MXY2Zm5lNXE2bGx6dHNzYmF5MGZudGoyZnR4aCZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/3oEjHV0z8S7WM4MwnK/giphy.gif)

---

**Thermal Control & Power Save**

Monitors system temps and scales load before the machine hits thermal throttle. Keeps sessions stable during extended use. Power Save mode lets you run overnight automation without destroying your power bill.

![Thermal](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExZjFkbWd0eHh2MHN5cmIyb245aHdxdGl6cGVld3c0MXB2bzRlemJzMCZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/26tn33aiTi1jkl6H6/giphy.gif)

---

**Secure Boot & Virtual GPU**

The VM boots into a sandboxed environment. Nothing leaks to the host. The Virtual GPU handles graphics inside the sandbox without touching your physical GPU drivers — useful when you're running multiple instances and don't want them stepping on each other.

![Secure](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExaXkzZDJidDZya3Y5N2k4eGh0NDVlN3Z3YzlsdDRleTcyZG9oMGdzYyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/xT9IgG50Lg7rusyiqU/giphy.gif)

### <img src="https://img.icons8.com/fluency/24/network.png" width="18"/> Network & Media

![Network Burst](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNXl4MW54aXZhZmphNHQzMndqaGEycXE5cXRhcHhzdm1sZm1ieGV2eSZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/QvMlVkJ3XSSj9cOxDM/giphy.gif)

Network and media tools are built in. No third-party capture software, no manual proxy setup.

![Signal](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExczN0a3FxMXZzMGc5Z2I3Mm54NjFvbzlzOGJsb29zaHNxbDBrYzl2YiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/KAq5w47R9rmTuxXUOs/giphy.gif)

**Integrated VPN & Proxy**

Routes the emulator's traffic through configurable endpoints. Works globally. Geo-restricted content, region-locked games, IP rotation for multi-account work — all handled inside the same interface.

![VPN](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExZzFsZTZlMm1rMHBpZjY4aGFuc2cwY21mNGV5bHd1dnE0cWo1ZW11YyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/3oEjHV0z8S7WM4MwnK/giphy.gif)

---

**Bandwidth Control**

Sets traffic priority so game packets go first. Everything else — background syncs, telemetry, whatever — gets deprioritized. The difference shows most in sessions where multiple instances are running simultaneously.

![Bandwidth](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNnUwcTBsOHY5MXZneDhzMjFsMW1rdXEwaWNpaW80MHkzZ3NxYm81NiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/du3J3cXyzhj75IOgvA/giphy.gif)

---

**Screen Recorder (60FPS)**

Captures at up to 4K/60fps with built-in trimming. No OBS, no external capture card. The recorder runs inside the VM so you get a clean signal without capture card overhead or desktop compositor artifacts.

![Recording](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNXpzN3l4c2N3aXUxZm9uNjFxcGZpdjE1NXo3NzJqdzZqcWFhZ3lsZiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/ule4vhcY1xEKQ/giphy.gif)

---

**Cloud Sync**

Pushes your profile, settings, and session data to the cloud. Switch machines mid-session and pick up where you left off. Everything — keybinds, macro scripts, instance configs — follows you.

![Cloud](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExaTZyMGo5cWZhZnB3MXY2Zm5lNXE2bGx6dHNzYmF5MGZudGoyZnR4aCZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/3oEjHV0z8S7WM4MwnK/giphy.gif)

## Project Layout

![Folder Structure](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExZjFkbWd0eHh2MHN5cmIyb245aHdxdGl6cGVld3c0MXB2bzRlemJzMCZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/26tn33aiTi1jkl6H6/giphy.gif)

```
src/         Core client — VM engine, UI layer, hardware Bridge
modules/     100+ feature implementations across 5 domains
web/         Landing page and documentation portal
config/      System-wide settings via emulator.cfg
```

![File Tree](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExaXkzZDJidDZya3Y5N2k4eGh0NDVlN3Z3YzlsdDRleTcyZG9oMGdzYyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/xT9IgG50Lg7rusyiqU/giphy.gif)

The `src/` directory is the VM core — no bloat, no demo code, no comments by policy. `modules/` is where the 100+ features live, each one self-contained. `config/` holds `emulator.cfg`, the single file that controls system-wide behavior.

![Structure Pulse](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNXl4MW54aXZhZmphNHQzMndqaGEycXE5cXRhcHhzdm1sZm1ieGV2eSZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/QvMlVkJ3XSSj9cOxDM/giphy.gif)
